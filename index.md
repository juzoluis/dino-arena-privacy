# Política de Privacidade — Dino Arena

**Última atualização: 4 de julho de 2026**

**Dino Arena** é um jogo para Android desenvolvido por Juzo Luis Hatao Marinelli
("nós"). Esta política descreve como o jogo trata as suas informações.

## Resumo: não coletamos nenhum dado

O Dino Arena é um jogo **100% offline**:

- **Não coletamos, armazenamos nem compartilhamos** nenhum dado pessoal
- **Não exige conta ou login** de nenhum tipo
- **Não contém anúncios**
- **Não contém compras** dentro do app
- **Não usa serviços de análise** (analytics) nem SDKs de terceiros que coletem dados
- **Não acessa** contatos, localização, câmera, microfone ou arquivos pessoais

## Dados salvos no seu aparelho

O progresso do jogo (dinossauros colecionados, moedas, configurações) fica salvo
**apenas localmente**, no armazenamento privado do app no seu aparelho. Nós não
temos acesso a esses dados.

Se o **backup do Android** estiver ativado no aparelho, o sistema operacional pode
copiar esse progresso para o Google Drive **da própria pessoa usuária**, para
restauração em caso de troca de aparelho. Esse backup é um recurso do Android,
controlado nas configurações do aparelho — nós não temos acesso a ele.

## Crianças

O Dino Arena é adequado para **todas as idades**. Como não coletamos nenhum dado,
nenhuma informação de crianças é coletada, armazenada ou compartilhada.

## Permissões

O jogo pede apenas as permissões mínimas necessárias para funcionar. Permissão de
**notificações** (se solicitada) é opcional e usada somente para avisos locais do
próprio jogo — nada é enviado pela internet.

## Mudanças nesta política

Se esta política mudar (por exemplo, em versões futuras com novos recursos), a
versão atualizada será publicada nesta mesma página, com nova data.

## Contato

Dúvidas sobre esta política: **juzoluis@gmail.com**

# Privacy Policy — Dino Arena (English)

**Last updated: July 4, 2026**

**Dino Arena** is an Android game developed by Juzo Luis Hatao Marinelli ("we").
This policy describes how the game handles your information.

## Summary: we do not collect any data

Dino Arena is a **fully offline** game:

- We **do not collect, store, or share** any personal data
- **No account or login** of any kind is required
- The game contains **no ads**
- The game contains **no in-app purchases**
- We use **no analytics services** or third-party SDKs that collect data
- The game does **not access** contacts, location, camera, microphone, or personal files

## Data saved on your device

Game progress (collected dinosaurs, coins, settings) is stored **locally only**,
in the app's private storage on your device. We have no access to this data.

If **Android Backup** is enabled on the device, the operating system may copy this
progress to the **user's own** Google Drive so it can be restored on a new device.
This backup is an Android feature controlled in the device settings — we have no
access to it.

## Children

Dino Arena is suitable for **all ages**. Since we collect no data at all, no
information from children is collected, stored, or shared.

## Permissions

The game requests only the minimum permissions needed to run. The **notifications**
permission (if requested) is optional and used solely for local, in-game reminders —
nothing is sent over the internet.

## Changes to this policy

If this policy changes (for example, in future versions with new features), the
updated version will be published on this same page with a new date.
